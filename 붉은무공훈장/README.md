# 붉은 무공훈장 : The Weight of Courage

스티븐 크레인의 소설 『붉은 무공훈장(The Red Badge of Courage)』을 배경으로 한
**미연시(비주얼 노벨) 스타일 게임**입니다.

미연시의 문법(① 선택지 분기 스토리 ② 스탯 게이지 ③ 멀티 엔딩
④ 캐릭터 초상화 · 이벤트 카드 ⑤ 대사창 연출)을 그대로 가져오되,
로맨스의 '호감도' 대신 아래 스탯 시스템으로, 공략 대상 대신
**전우(헨리 · 짐 · 야영지의 병사들)와의 유대**를 다룹니다.

## 스탯 시스템

우측 상단 게이지 패널(그룹별)로 항상 표시됩니다. (`TAB` 으로 On/Off)

| 스탯 | 초기값 | 의미 |
|------|:---:|------|
| 신뢰도 (Trust) | 50 | 헨리가 마음을 여는 정도. 40↓면 거짓말 시작 |
| 공감 (Empathy) | 50 | **핵심.** 헨리를 이해하려 했는가 |
| 인간 본능 (Humanity) | 50 | **주제.** 살고 싶은 인간의 본능 |
| 사회적 역할 (Duty) | 50 | "군인은 이래야 한다" (인간 본능과 반비례 아님) |
| 죄책감 (Guilt) | 40 | 80↑면 무모한 돌격 → 배드 위험 |
| 용기 (Courage) | 10 | 마지막에만 중요, 내려가지 않음 |
| 전쟁 체험도 | 0 | (숨김) 시체·부상·죽음을 얼마나 목격했는가 |

### 갈등의 저울 (Conflict Scale)
화면 좌측 상단에 항상 표시. 선택할 때마다 **인간으로 살아남기 ⚖ 군인으로
살아가기** 가 움직이며, 엔딩의 최종 방향에 반영됩니다.

### 기억 조각 (아이템)
`군복 · 짐의 군번줄 · 붉은 손수건 · 탄피 · 마지막 깃발` — 모두 모으고 공감 90↑
이면 **히든 엔딩**. (Ch0~2 에서는 군복·군번줄 획득 가능)

### 엔딩 (5종 — 마지막 장면이 완전히 다름)
판정 우선순위 : HIDDEN → TRUE → BAD → GOOD → NORMAL

| 엔딩 | 조건 | 마지막 장면 |
|------|------|------|
| **HIDDEN** 「인간이라는 이름」 | 기억 조각 5개 전부 + 공감 90↑ | 없던 문장 등장, 조각이 책갈피가 됨 |
| **TRUE** 「진정한 용기」 | 공감 80↑·신뢰 80↑·죄책감 40↓·용기 80↑ | 헨리가 웃으며 "고마워, 이제 나를 미워하지 않아" |
| **BAD** 「붉은 무공훈장」 | 신뢰 30↓ 또는 죄책감 80↑ | 헨리가 등 돌림, 제목이 피처럼 붉게 변하는 CG |
| **GOOD** 「앞으로 나아간 사람」 | 공감 60↑·용기 60↑ | 말 적은 헨리의 뒷모습, "그래도 걸어갈 것이다" |
| **NORMAL** 「정답 없는 질문」 | 그 외 | 도서관, 책은 그대로, "아직 답을 못 찾았다" |

> 조각을 모두 모으고 공감이 높으면 HIDDEN이 우선합니다(수집 보상 엔딩).
> TRUE는 조각을 덜 모으면서 용기·신뢰를 끝까지 올린 플레이에서 열립니다.
> 엔딩 후 "당신에게 용기란 무엇입니까?" 마지막 질문 화면이 나옵니다(저장 안 됨).

## 실행 방법

```bash
pip install pygame-ce        # 최초 1회
python game.py
```

- `마우스 클릭` / `Space` / `Enter` : 대사 진행 · 선택
- `TAB` : 스탯 패널 On/Off
- `ESC` : 종료

## 파일 구성

- `game.py` — 비주얼 노벨 엔진 (렌더링 · 선택지 · 스탯 · 탐색 · 엔딩)
- `story.py` — 시나리오 데이터 (Chapter 0~2). **여기만 고치면 내용이 바뀝니다.**
- `assets/` — 이미지 폴더

## 수록 내용

- **Chapter 0** 프롤로그 — 책 속으로
- **Chapter 1** 전투 전날 밤 (야영지 자유 탐색: 모닥불·보급마차·강가)
- **Chapter 2** 도망친 병사 (첫 전투 · 도주 · 다람쥐 · 짐의 죽음)
- **Chapter 3** 붉은 무공훈장 (부상병 · 아군이 낸 상처 · 진실 고백 분기)
- **Chapter 4** 용기의 의미 (병사 구조 · 깃발 · 마지막 돌격)
- **Chapter 5** 당신이라면 어떤 선택을 하겠습니까 (현실 복귀 → 엔딩 분기)
- 조건부 이벤트(신뢰·죄책감·공감 임계값), Chapter Result 화면, 기억 조각 5종
- **엔딩 5종**(장면 완전 분리) + 마지막 질문 화면
